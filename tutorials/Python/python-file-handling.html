Python File Handling

File handling lets your programs read from and write to files on disk.

The builtâ€‘in open() function returns a file object you can use to read, write, or append data.

Opening and reading files

# Read entire file
with open("notes.txt", "r", encoding="utf-8") as f:
    content = f.read()
    print(content)

# Read line by line
with open("notes.txt", "r", encoding="utf-8") as f:
    for line in f:
        print(line.strip())

The with statement automatically closes the file when the block ends, even if errors occur.

Writing and appending

# Write (overwrite) file
with open("output.txt", "w", encoding="utf-8") as f:
    f.write("First line\n")
    f.write("Second line\n")

# Append to existing file
with open("output.txt", "a", encoding="utf-8") as f:
    f.write("Appended line\n")

Use mode "w" to create/overwrite a file, "a" to append, and "r" to read.

Working with paths

from pathlib import Path

base_dir = Path(__file__).parent
data_file = base_dir / "data" / "users.csv"

with data_file.open("r", encoding="utf-8") as f:
    for line in f:
        print(line.strip().split(","))
