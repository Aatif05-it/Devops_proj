Python Pandas Basics

Pandas provides Series and DataFrame structures for working with tabular data.

It makes loading, cleaning, and analyzing data from CSV, Excel, and databases much easier.

Creating a DataFrame

import pandas as pd

data = {
    "name": ["Alice", "Bob", "Cara"],
    "age": [25, 30, 22],
    "city": ["Mumbai", "Delhi", "Pune"],
}

df = pd.DataFrame(data)
print(df)

DataFrame is like a table with labeled columns and rows.

Loading from CSV and basic operations

df = pd.read_csv("users.csv")

print(df.head())           # first 5 rows
print(df["age"].mean())    # average age

filtered = df[df["age"] >= 25]
print(filtered[["name", "city"]])

Pandas lets you filter rows, select columns, and compute statistics in just a few lines.
