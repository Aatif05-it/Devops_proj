Python Classes & Objects

Classes define blueprints for objects, bundling data (attributes) and behavior (methods) together.

Objects are instances of classes, each with its own copy of the classâ€™s attributes.

Defining a simple class

class Student:
    # constructor
    def __init__(self, name, age):
        self.name = name      # instance attribute
        self.age = age

    # instance method
    def introduce(self):
        print(f"My name is {self.name} and I am {self.age} years old")

# create objects
s1 = Student("Alice", 20)
s2 = Student("Ravi", 22)

s1.introduce()
s2.introduce()

The __init__ method runs automatically when you create a new object and sets up initial state.

Class vs instance attributes

class Counter:
    count = 0   # class attribute (shared)

    def __init__(self):
        Counter.count += 1

c1 = Counter()
c2 = Counter()
print(Counter.count)  # 2

Class attributes are shared across all instances, while instance attributes live on each object separately.
