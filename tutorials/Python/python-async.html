Python Async/Await

Async/await enables asynchronous, nonâ€‘blocking code using coroutines and the asyncio event loop.

It is ideal for many concurrent I/O operations such as HTTP calls or database queries.

Basic async function

import asyncio

async def say_after(delay, message):
    await asyncio.sleep(delay)
    print(message)

async def main():
    await say_after(1, "Hello")
    await say_after(1, "World")

asyncio.run(main())

async defines a coroutine; await pauses it until the awaited operation completes.

Running tasks concurrently

import asyncio

async def worker(name, delay):
    print(name, "started")
    await asyncio.sleep(delay)
    print(name, "finished")

async def main():
    tasks = [
        asyncio.create_task(worker("A", 2)),
        asyncio.create_task(worker("B", 2)),
    ]
    await asyncio.gather(*tasks)

asyncio.run(main())

create_task schedules coroutines to run concurrently on the same event loop.
