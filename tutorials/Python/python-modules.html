Python Modules & Packages

Modules are Python files that contain code (functions, classes, variables) you can import into other files.

Packages are folders that group related modules together, usually with an __init__.py file inside.

Importing builtâ€‘in modules

import math

print(math.pi)
print(math.sqrt(16))

from random import randint

print(randint(1, 10))

Use import module to bring in the whole module, or from module import name to import specific functions or classes.

Creating your own module

# File: utils.py
def add(a, b):
    return a + b

def greet(name):
    return f"Hello, {name}!"

# File: main.py (same folder)
import utils

print(utils.add(2, 3))
print(utils.greet("Alice"))

Any .py file can act as a module; just import it by its filename (without the .py extension).

Basic package structure

# Folder structure:
# myapp/
#   calculations/
#     __init__.py
#     add.py
#     subtract.py

# add.py
def add(a, b):
    return a + b

# subtract.py
def subtract(a, b):
    return a - b

# main.py
from calculations.add import add
from calculations.subtract import subtract

print(add(5, 3))
print(subtract(5, 3))

Putting an __init__.py file in a folder makes it a package, allowing dotâ€‘notation imports like package.module.
