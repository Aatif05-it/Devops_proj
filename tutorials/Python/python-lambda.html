Python Lambda Functions

Lambda functions are small anonymous functions defined with the lambda keyword.

They are useful for short oneâ€‘line operations, especially when used with map, filter, or sorted.

Basic lambda syntax

# Regular function
def square(x):
    return x * x

# Equivalent lambda
square_lambda = lambda x: x * x

print(square(5))
print(square_lambda(5))

A lambda expression has arguments on the left and a single expression on the right: lambda args: expression.

Using lambda with map()

numbers = [1, 2, 3, 4]

squares = list(map(lambda n: n * n, numbers))
print(squares)   # [1, 4, 9, 16]

map() applies the lambda to every item in the iterable and returns a new iterator.

Using lambda with filter() and sorted()

nums = [5, 1, 8, 3, 2]

evens = list(filter(lambda n: n % 2 == 0, nums))
print(evens)  # [8, 2]

students = [
    {"name": "Alice", "score": 92},
    {"name": "Bob", "score": 75},
    {"name": "Cara", "score": 88},
]

# sort by score descending
students_sorted = sorted(students, key=lambda s: s["score"], reverse=True)
print(students_sorted)

Lambdas make it easy to define quick sort keys and filter conditions without creating full function definitions.
