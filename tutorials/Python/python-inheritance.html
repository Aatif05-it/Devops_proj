Python Inheritance

Inheritance lets a child class reuse and extend behavior from a parent class.

This supports code reuse and models â€œisâ€‘aâ€ relationships such as Student is a Person.

Single inheritance

class Person:
    def __init__(self, name):
        self.name = name

    def introduce(self):
        print(f"Hi, I am {self.name}")

class Student(Person):          # Student inherits Person
    def __init__(self, name, roll_no):
        super().__init__(name)  # call parent constructor
        self.roll_no = roll_no

    def details(self):
        print(f"{self.name} → Roll no: {self.roll_no}")

s = Student("Alice", 101)
s.introduce()   # from Person
s.details()     # from Student

Use super() to call parent methods like __init__ from the child class.

Multiple inheritance

class Loggable:
    def log(self, message):
        print("[LOG]", message)

class Timestamped:
    def timestamp(self):
        import datetime
        return datetime.datetime.now()

class Service(Loggable, Timestamped):
    def process(self):
        self.log(f"Started at {self.timestamp()}")

service = Service()
service.process()

Python supports multiple inheritance; the method resolution order (MRO) decides which parent method is used when names collide.
