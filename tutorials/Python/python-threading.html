Python Multithreading

Multithreading lets you run multiple tasks seemingly at the same time within a single process.

In CPython it is best used for I/Oâ€‘bound tasks like network requests or file operations.

Creating basic threads

import threading
import time

def worker(name):
    print(f"{name} starting")
    time.sleep(1)
    print(f"{name} done")

t1 = threading.Thread(target=worker, args=("Thread 1",))
t2 = threading.Thread(target=worker, args=("Thread 2",))

t1.start()
t2.start()

t1.join()
t2.join()

print("All threads finished")

start() begins the threadâ€™s work, and join() waits for it to finish before continuing.

Using Lock for shared data

import threading

counter = 0
lock = threading.Lock()

def increment():
    global counter
    for _ in range(100000):
        with lock:
            counter += 1

t1 = threading.Thread(target=increment)
t2 = threading.Thread(target=increment)

t1.start(); t2.start()
t1.join(); t2.join()

print("Counter:", counter)

A Lock ensures only one thread at a time updates shared state, preventing race conditions.
