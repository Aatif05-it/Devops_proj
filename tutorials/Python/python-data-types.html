Numeric and string examples

# Numeric types
age = 21              # int
pi = 3.14159          # float
z = 2 + 3j            # complex

# String type
name = "Alice"
message = "Hello, " + name

print(type(age))      # <class 'int'>
print(type(pi))       # <class 'float'>
print(type(z))        # <class 'complex'>
print(type(name))     # <class 'str'>

The type() function shows the class of any object, which is helpful when learning types.

Sequence, mapping, and set examples

# List (mutable sequence)
numbers = [1, 2, 3]

# Tuple (immutable sequence)
point = (10, 20)

# Range (sequence of integers)
evens = range(0, 10, 2)

# Dictionary (mapping)
user = {"name": "Alice", "age": 21}

# Set (unique items, unordered)
tags = {"python", "backend", "web"}

print(len(numbers))  # 3
print(user["name"])  # Alice
print("python" in tags)  # True

Lists and dictionaries are the most common structures for collections, while sets are great for uniqueness and membership tests.

Type conversion (casting)

num_str = "42"
num = int(num_str)        # "42" → 42
pi_str = "3.14"
pi = float(pi_str)        # "3.14" → 3.14

flag = bool(0)            # False
text = str(123)           # "123"

print(num, type(num))
print(pi, type(pi))
print(flag, type(flag))
print(text, type(text))

Constructor functions like int(), float(), str(), and bool() convert between compatible types.
