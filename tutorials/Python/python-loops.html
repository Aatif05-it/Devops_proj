Python For & While Loops

Loops let you repeat a block of code multiple times. Python has two main loop types: for and while.

Use for loops when you know how many items to iterate; use while loops when you loop until a condition changes.

For loop over sequences

fruits = ["apple", "banana", "cherry"]

for fruit in fruits:
    print(fruit)

# Loop with index using enumerate()
for index, fruit in enumerate(fruits):
    print(index, fruit)

The for loop iterates directly over items in a list, string, or other iterable.

Using range()

# 0 to 4
for i in range(5):
    print(i)

# 1 to 10, step 2
for i in range(1, 11, 2):
    print(i)

range() generates a sequence of numbers and is commonly used in for loops.

While loop basics

count = 1

while count <= 5:
    print("Count:", count)
    count += 1

A while loop runs as long as its condition stays true. Always make sure something changes inside to avoid infinite loops.

break and continue

# break: stop the loop early
for i in range(1, 10):
    if i == 5:
        break
    print(i)

# continue: skip this iteration
for i in range(1, 6):
    if i == 3:
        continue
    print(i)

break exits the loop completely; continue jumps to the next iteration.
