Python NumPy Basics

NumPy provides fast nâ€‘dimensional arrays and vectorized operations for numerical computing.

It is the foundation of many data science and machine learning libraries.

Creating arrays

import numpy as np

a = np.array([1, 2, 3])
b = np.array([[1, 2, 3],
              [4, 5, 6]])

print(a.shape)   # (3,)
print(b.shape)   # (2, 3)

Unlike Python lists, NumPy arrays have a fixed type and support efficient vectorized math.

Vectorized operations

x = np.array([1, 2, 3])
y = np.array([10, 20, 30])

print(x + y)      # [11 22 33]
print(x * 10)     # [10 20 30]
print(np.mean(y)) # 20.0

Operations on arrays apply elementâ€‘wise without explicit loops, making code shorter and faster.
