Python List Comprehensions

List comprehensions provide a concise way to create lists from iterables.

They often replace simple forâ€‘loops with a single readable expression.

Basic syntax

numbers = [1, 2, 3, 4]
squares = [n * n for n in numbers]

print(squares)  # [1, 4, 9, 16]

The pattern is [expression for item in iterable], which builds a new list.

With condition (filtering)

nums = [1, 2, 3, 4, 5, 6, 7, 8]
evens = [n for n in nums if n % 2 == 0]

print(evens)  # [2, 4, 6, 8]

You can add an if clause to keep only items that match a condition.

Nested comprehensions

matrix = [[1, 2, 3],
          [4, 5, 6]]

flattened = [num for row in matrix for num in row]
print(flattened)  # [1, 2, 3, 4, 5, 6]

Multiple for clauses support nested loops and flattening structures.
