Python Lists & Tuples

Lists are mutable sequences, while tuples are immutable sequences used for fixed collections of items.

Both support indexing, slicing, iteration, and many of the same functions like len(), min(), max(), and sum().

Creating lists and tuples

# List (mutable)
fruits = ["apple", "banana", "cherry"]

# Tuple (immutable)
point = (10, 20)

print(fruits[0])   # apple
print(point[1])    # 20

Use lists when you need to modify contents; use tuples when data should not change or to represent fixed records.

Common list operations

numbers = [10, 20, 30]

numbers.append(40)        # add to end
numbers.insert(1, 15)     # insert at index 1
numbers.remove(20)        # remove first 20
popped = numbers.pop()    # remove last element

print(numbers)            # [10, 15, 30]
print(popped)             # 40

Methods like append(), insert(), remove(), and pop() make lists flexible for dynamic data.

Slicing and iteration

nums = [1, 2, 3, 4, 5]

print(nums[1:4])    # [2, 3, 4]
print(nums[:3])     # [1, 2, 3]
print(nums[::2])    # [1, 3, 5]

for n in nums:
    print(n)

Slices create new lists from parts of an existing sequence, while for loops are the standard way to traverse items.

Converting between list and tuple

t = (1, 2, 3)
lst = list(t)       # tuple → list
lst.append(4)

back_to_tuple = tuple(lst)  # list → tuple

print(t)             # (1, 2, 3)
print(lst)           # [1, 2, 3, 4]
print(back_to_tuple) # (1, 2, 3, 4)

Converting tuples to lists allows modification, while converting back to tuples freezes the data again.
