Python Operators

Operators are symbols that perform operations on values and variables, such as arithmetic, comparison, and logical checks.

Python groups operators into categories: arithmetic, assignment, comparison, logical, bitwise, identity, and membership.

Arithmetic operators

a = 15
b = 4

print("a + b =", a + b)   # 19
print("a - b =", a - b)   # 11
print("a * b =", a * b)   # 60
print("a / b =", a / b)   # 3.75 (float division)
print("a // b =", a // b) # 3   (floor division)
print("a % b =", a % b)   # 3
print("a ** b =", a ** b) # 15^4

Use // for integer division and ** for exponentiation, which often replace manual math.

Comparison and logical operators

x = 10
y = 20

print(x == y)   # False
print(x != y)   # True
print(x < y)    # True
print(x >= 5)   # True

is_adult = True
has_ticket = False

print(is_adult and has_ticket)  # False
print(is_adult or has_ticket)   # True
print(not is_adult)             # False

Comparison operators return booleans, which chain naturally with and, or, and not in conditions.

Assignment and augmented assignment

count = 0      # simple assignment
count += 1     # count = count + 1
count *= 2     # count = count * 2
count -= 3     # count = count - 3

print(count)

Augmented assignments like += and *= update a variable in place and make code shorter.

Identity and membership

nums = [1, 2, 3]
a = nums
b = [1, 2, 3]

print(a is nums)       # True (same object)
print(b is nums)       # False (different list)
print(a == b)          # True (same contents)

print(2 in nums)       # True
print(5 not in nums)   # True

is checks whether two variables reference the same object, while == compares values; in tests membership inside sequences.
