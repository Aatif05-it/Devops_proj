Python Generators

Generators are functions that yield a sequence of values lazily instead of returning them all at once.

They help with memoryâ€‘efficient iteration over large or infinite sequences.

Creating a generator function

def countdown(n):
    while n > 0:
        yield n
        n -= 1

for num in countdown(3):
    print(num)

The yield keyword pauses the function and sends a value back to the caller each time.

Using next() with generators

gen = countdown(3)

print(next(gen))  # 3
print(next(gen))  # 2
print(next(gen))  # 1
# next(gen) would now raise StopIteration

Generators are iterators, so you can manually advance them with next().

Generator expressions

squares = (n * n for n in range(5))

for value in squares:
    print(value)

Generator expressions look like list comprehensions but use parentheses and produce values on demand.
