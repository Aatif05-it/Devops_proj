Python Introduction

Python is a high-level, interpreted programming language known for its simplicity and readability.

Created by Guido van Rossum in 1991, Python emphasizes code readability with its clean syntax.

Hello, World!

Your first Python program prints "Hello, World!" to the console.

# First Python program
print("Hello, World!")
print("Welcome to CodeNexa Academy")

Run with: python filename.py

Python versions

  • Python 2.x (legacy, end-of-life)
  • Python 3.x (current, recommended)
# Check your version
python --version
# or
python3 --version
