Python Strings

Strings represent textual data and are sequences of Unicode characters.

You can create them with single quotes, double quotes, or triple quotes for multiâ€‘line text.

Creating and accessing strings

single = 'Hello'
double = "World"
multi = """This is
a multi-line string."""

message = single + " " + double
print(message)        # Hello World
print(message[0])     # 'H'
print(message[-1])    # 'd'

Strings are zeroâ€‘indexed, and negative indexes count from the end.

Slicing strings

text = "Python Programming"

print(text[0:6])     # 'Python'
print(text[7:])      # 'Programming'
print(text[:6])      # 'Python'
print(text[::2])     # 'Pto rgamn'

Slices let you extract substrings using the pattern [start:stop:step].

Useful string methods

name = "  alice  "

print(name.upper())          # '  ALICE  '
print(name.strip())          # 'alice'
print(name.replace("a", "A"))# '  Alice  '

sentence = "python is awesome"
print(sentence.title())      # 'Python Is Awesome'
print(sentence.split())      # ['python', 'is', 'awesome']

Methods like upper(), lower(), strip(), replace(), and split() cover most daily string cleaning needs.
