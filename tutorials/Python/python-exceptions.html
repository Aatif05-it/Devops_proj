tryâ€‘exceptâ€‘elseâ€‘finally

try:
    f = open("data.txt", "r", encoding="utf-8")
except FileNotFoundError:
    print("File not found.")
else:
    print("File opened successfully.")
    print(f.read())
    f.close()
finally:
    print("This always runs.")

else runs only if no exception happens; finally runs whether there was an exception or not.

Raising your own exceptions

def withdraw(balance, amount):
    if amount > balance:
        raise ValueError("Insufficient funds")
    return balance - amount

try:
    remaining = withdraw(1000, 1500)
except ValueError as e:
    print("Error:", e)

Use raise to signal that something went wrong according to your own business rules.
