Python Functions

Functions are reusable blocks of code defined with def that can accept parameters and return values.

They help you organize code, avoid repetition, and make programs easier to test.

Defining and calling a function

def greet(name):
    """Return a greeting message."""
    return f"Hello, {name}!"

message = greet("Alice")
print(message)

Use return to send a value back to the caller; without return, the function returns None.

Default and keyword arguments

def introduce(name, country="India"):
    print(f"My name is {name} and I am from {country}")

introduce("Ravi")
introduce("Sara", country="USA")

Default parameters provide fallback values; keyword arguments make calls more readable.

*args and **kwargs

def add_all(*numbers):
    total = 0
    for n in numbers:
        total += n
    return total

def print_user(**info):
    for key, value in info.items():
        print(key, ":", value)

print(add_all(1, 2, 3, 4))
print_user(name="Alice", age=21, city="Mumbai")

*args collects extra positional arguments into a tuple, while **kwargs collects keyword arguments into a dictionary.
