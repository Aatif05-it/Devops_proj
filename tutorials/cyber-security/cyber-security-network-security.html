Network Security

Network security protects systems, devices, and data as information moves across local and public networks.

Core Controls

- Segment networks (VLANs) to reduce lateral movement
- Allow only required inbound ports
- Monitor traffic logs and alert on anomalies
- Patch routers, firewalls, and network services

What You Will Learn

  • How network segmentation reduces attack spread.
  • How firewall rules, IDS/IPS, and monitoring work together.
  • How to build a basic defense-in-depth network model for CNS labs.

Practice Task

Design a small office network with one public zone, one internal zone, and one restricted admin zone, then define allowed traffic between them.
