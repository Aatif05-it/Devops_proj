Cryptography Basics

Cryptography protects confidentiality, integrity, and authenticity using encryption, hashing, and digital signatures.

Example

# Hashing for password storage (concept)
password_hash = HASH(password + salt)

# Symmetric encryption for data-at-rest
ciphertext = AES_Encrypt(secret_key, plaintext)

What You Will Learn

  • Difference between symmetric and asymmetric encryption.
  • When to use hashing, salting, and key derivation.
  • How TLS certificates help secure client-server communication.

Practice Task

Create a table mapping security goals to controls: confidentiality to encryption, integrity to hashes, and authenticity to digital signatures.
