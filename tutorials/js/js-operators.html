JS Operators

Operators are symbols that let JavaScript perform arithmetic, compare values, assign variables, and combine logical conditions.

Arithmetic and comparison

This example uses arithmetic operators to add numbers and a comparison operator to check if the result is greater than 10. 

<!DOCTYPE html>
<html>
<body>

  <p id="result"></p>

  <script>
    let a = 4;
    let b = 9;

    let sum = a + b;       // arithmetic +
    let isBig = sum > 10;  // comparison >

    const text =
      "Sum: " + sum + "\\n" +
      "Is sum > 10? " + isBig;

    document.getElementById("result").textContent = text;
  </script>

</body>
</html>

JavaScript groups operators into arithmetic, assignment, comparison, logical, and more specialized types like bitwise and ternary operators.
