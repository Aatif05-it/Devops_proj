JS Local Storage

Local storage allows web pages to store key–value pairs in the browser that persist across reloads.

Save and load a value

This example saves a username in localStorage and reads it back when the script runs. 

<!DOCTYPE html>
<html>
<body>

  <p id="greeting"></p>

  <script>
    // Save value
    localStorage.setItem("username", "Alex");

    // Read value
    const name = localStorage.getItem("username");

    document.getElementById("greeting").textContent =
      "Hello, " + name + " (from localStorage)";
  </script>

</body>
</html>

Because local storage only stores strings, complex objects should be converted with JSON.stringify() and JSON.parse().
