JS Syntax

JavaScript syntax is the set of rules for how programs are written using statements, expressions, and identifiers.

Basic statements

This example declares variables and uses an expression to calculate a total which is then shown in the page. 

<!DOCTYPE html>
<html>
<body>

  <p id="result"></p>

  <script>
    let a = 5;
    let b = 7;
    let c = a + b; // statement ending with ;

    document.getElementById("result").textContent =
      "Result: " + c;
  </script>

</body>
</html>

Semicolons are optional in many places but recommended to clearly end each statement.
