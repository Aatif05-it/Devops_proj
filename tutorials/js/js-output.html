JS Output

JavaScript can show output in different ways: writing into HTML, showing alert boxes, and logging to the console.

Basic output methods

This example uses innerHTML, alert(), and console.log() to display messages. 

<!DOCTYPE html>
<html>
<body>

  <p id="demo"></p>

  <script>
    const message = "Hello from JavaScript!";

    // Write into HTML
    document.getElementById("demo").innerHTML = message;

    // Alert box
    alert(message);

    // Console
    console.log(message);
  </script>

</body>
</html>

For debugging, console.log() is often the most useful output method because it does not interrupt the user.
