JS Arrow Functions

Arrow functions provide a shorter syntax for functions and capture this from the surrounding scope.

Arrow vs regular function

This example compares a regular function and an arrow function used with map(). 

<!DOCTYPE html>
<html>
<body>

  <pre id="output"></pre>

  <script>
    const numbers = [1, 2, 3];

    const double1 = numbers.map(function (n) {
      return n * 2;
    });

    const double2 = numbers.map(n => n * 2);

    const text =
      "Regular function: " + double1.join(", ") + "\\n" +
      "Arrow function: " + double2.join(", ");

    document.getElementById("output").textContent = text;
  </script>

</body>
</html>

Arrow functions are ideal for short callbacks; avoid them as object methods when you need a dynamic this.
