JS DOM Styling

JavaScript can change styles and classes to update how elements look without reloading the page.

Toggle a CSS class

This example toggles a highlight class on a paragraph when a button is clicked. 

<!DOCTYPE html>
<html>
<head>
  <style>
    .highlight {
      color: #ffffff;
      background-color: #04aa6d;
      padding: 4px 8px;
    }
  </style>
</head>
<body>

  <p id="text">Click the button to highlight this text.</p>
  <button id="toggle">Toggle highlight</button>

  <script>
    const text = document.getElementById("text");
    const toggleBtn = document.getElementById("toggle");

    toggleBtn.addEventListener("click", function () {
      text.classList.toggle("highlight");
    });
  </script>

</body>
</html>

The classList API supports methods like add, remove, and toggle to manage multiple CSS classes.
