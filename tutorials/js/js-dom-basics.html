JS DOM Basics

The Document Object Model (DOM) represents the HTML page as a tree of nodes that JavaScript can read and change.

getElementById example

This example selects a paragraph by its id and changes its text when a button is clicked. 

<!DOCTYPE html>
<html>
<body>

  <p id="demo">Original text.</p>
  <button onclick="changeText()">Change</button>

  <script>
    function changeText() {
      const p = document.getElementById("demo");
      p.textContent = "Text updated using the DOM!";
    }
  </script>

</body>
</html>

Methods like document.getElementById() return a reference to an element so you can read or modify its properties.
