JS Where To

JavaScript code can be placed inside the HTML page or in an external .js file that you link with a <script> tag.

Script in head

This example places the script in the <head> and runs a function when the button is clicked. 

<!DOCTYPE html>
<html>
<head>
  <script>
    function changeText() {
      document.getElementById("demo").textContent = "Paragraph changed.";
    }
  </script>
</head>
<body>

  <h2>Demo JavaScript in Head</h2>
  <p id="demo">A paragraph.</p>
  <button onclick="changeText()">Try it</button>

</body>
</html>

You can also keep JavaScript in a separate file and include it with <script src="app.js"></script> for better organization.
