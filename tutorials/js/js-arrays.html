JS Arrays

An array stores multiple values in a single variable, such as a list of strings or numbers.

Create and loop an array

This example creates an array of fruits and uses forEach() to show them as a list. 

<!DOCTYPE html>
<html>
<body>

  <ul id="fruits"></ul>

  <script>
    const fruits = ["Apple", "Banana", "Mango"];

    const list = document.getElementById("fruits");

    fruits.forEach(function (fruit) {
      const li = document.createElement("li");
      li.textContent = fruit;
      list.appendChild(li);
    });
  </script>

</body>
</html>

Arrays come with many helpful methods like push, pop, map, and filter for working with collections.
