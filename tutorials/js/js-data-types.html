JS Data Types

JavaScript variables can hold different data types, including primitives like strings, numbers, and booleans, and reference types like objects.

Common types

This example declares several values of different types and prints their typeof results. 

<!DOCTYPE html>
<html>
<body>

  <pre id="info"></pre>

  <script>
    const name = "Alex";          // string
    const age = 21;               // number
    const isStudent = true;       // boolean
    let nickname;                 // undefined
    const score = null;           // null
    const user = { name: "Alex" } // object

    const output =
      "name: " + name + " (" + typeof name + ")" + "\\n" +
      "age: " + age + " (" + typeof age + ")" + "\\n" +
      "isStudent: " + isStudent + " (" + typeof isStudent + ")" + "\\n" +
      "nickname: " + nickname + " (" + typeof nickname + ")" + "\\n" +
      "score: " + score + " (" + typeof score + ")" + "\\n" +
      "user: " + JSON.stringify(user) + " (" + typeof user + ")";

    document.getElementById("info").textContent = output;
  </script>

</body>
</html>

Remember that typeof null returns "object" for historical reasons, even though null is a primitive value.
