JS Timers (setTimeout)

The setTimeout() function runs code once after a specified delay in milliseconds.

Delay a message

This example waits 2 seconds after clicking the button and then updates the text. 

<!DOCTYPE html>
<html>
<body>

  <p id="status">Click the button to start.</p>
  <button id="start">Start timer</button>

  <script>
    const statusEl = document.getElementById("status");
    const startBtn = document.getElementById("start");

    startBtn.addEventListener("click", function () {
      statusEl.textContent = "Waiting 2 seconds...";
      setTimeout(function () {
        statusEl.textContent = "Timer finished!";
      }, 2000);
    });
  </script>

</body>
</html>

Timers are useful for delays, animations, and temporary messages; use clearTimeout() if you need to cancel a scheduled action.
