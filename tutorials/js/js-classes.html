JS Classes

Classes are templates for creating objects with shared properties and methods, built on top of JavaScript prototypes.

Class and instance

This example defines a Person class with a constructor and method, then creates an instance. 

<!DOCTYPE html>
<html>
<body>

  <p id="info"></p>

  <script>
    class Person {
      constructor(name, age) {
        this.name = name;
        this.age = age;
      }

      describe() {
        return this.name + " is " + this.age + " years old.";
      }
    }

    const student = new Person("Alex", 21);
    document.getElementById("info").textContent = student.describe();
  </script>

</body>
</html>

Classes can extend other classes using extends and call parent constructors with super().
