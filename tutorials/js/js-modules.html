JS Modules

ES modules let you split code into files and share functions or values using export and import.

Simple module example

This example shows a small math.js module and an HTML file that imports it. 

<!-- math.js -->
export function add(a, b) {
  return a + b;
}

<!-- index.html -->
<!DOCTYPE html>
<html>
<body>

  <p id="result"></p>

  <script type="module">
    import { add } from "./math.js";

    const sum = add(2, 3);
    document.getElementById("result").textContent =
      "2 + 3 = " + sum;
  </script>

</body>
</html>

Modules load in strict mode and each module has its own scope, which helps avoid global variable conflicts.
