JS DOM

The Document Object Model (DOM) represents the HTML page as a tree of objects that JavaScript can read and change.

Basic example

This example changes the text inside a heading when a button is clicked, using document.getElementById(). 

<!DOCTYPE html>
<html>
<body>

  <h1 id="demo">This is some text.</h1>
  <button onclick="changeText()">Change Text</button>

  <script>
    function changeText() {
      const heading = document.getElementById("demo");
      heading.textContent = "Text changed by JavaScript!";
    }
  </script>

</body>
</html>

The DOM lets JavaScript find elements and update their content, attributes, and styles while the page is running.
