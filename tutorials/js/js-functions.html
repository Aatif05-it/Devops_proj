JS Functions

Functions group reusable code into a single block that can be called with different inputs to produce results.

Function declaration

This example defines a function that adds two numbers and then displays the result. 

<!DOCTYPE html>
<html>
<body>

  <p id="result"></p>

  <script>
    function add(a, b) {
      return a + b;
    }

    const sum = add(5, 7);

    document.getElementById("result").textContent =
      "5 + 7 = " + sum;
  </script>

</body>
</html>

Functions can take parameters, return values, and help keep code organized by avoiding duplication.
