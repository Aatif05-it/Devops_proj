JS Async & Await

async and await provide a cleaner way to work with promises, making asynchronous code look synchronous.

Async function example

This example wraps a delay promise in an async function and uses await to pause until it resolves. 

<!DOCTYPE html>
<html>
<body>

  <p id="status">Waiting...</p>

  <script>
    const statusEl = document.getElementById("status");

    function delay(ms) {
      return new Promise(resolve => setTimeout(resolve, ms));
    }

    async function run() {
      statusEl.textContent = "Running...";
      await delay(1000);
      statusEl.textContent = "Finished after 1 second";
    }

    run().catch(error => {
      statusEl.textContent = "Error: " + error;
    });
  </script>

</body>
</html>

Inside an async function, await can be used with any promise and errors can be handled with a normal try...catch block.
