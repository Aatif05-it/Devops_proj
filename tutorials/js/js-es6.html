JS ES6+ Features

ES6 and later versions introduced modern JavaScript features such as let/const, arrow functions, template literals, and destructuring.

Template literals and destructuring

This example uses object destructuring to extract properties and a template literal to build a message. 

<!DOCTYPE html>
<html>
<body>

  <pre id="info"></pre>

  <script>
    const user = { name: "Alex", city: "Mumbai", role: "Student" };

    const { name, city } = user;

    const message = `User ${name} lives in ${city}.`;

    document.getElementById("info").textContent = message;
  </script>

</body>
</html>

Other helpful ES6+ additions include default parameters, spread and rest syntax, classes, modules, and async functions.
