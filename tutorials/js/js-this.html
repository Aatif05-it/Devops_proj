JS This & Context

The value of this depends on how a function is called and usually refers to the current object.

this in a method

In an object method, this points to the object that owns the method. 

<!DOCTYPE html>
<html>
<body>

  <p id="info"></p>

  <script>
    const user = {
      name: "Alex",
      sayHello: function () {
        return "Hello, I am " + this.name;
      }
    };

    document.getElementById("info").textContent = user.sayHello();
  </script>

</body>
</html>

Arrow functions do not have their own this; they inherit it from the surrounding scope, which is useful in many callbacks.
