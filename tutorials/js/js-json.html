JS JSON

JSON (JavaScript Object Notation) is a text format for representing structured data, commonly used to send data between browser and server.

stringify and parse

This example converts an object into a JSON string with JSON.stringify() and back into an object with JSON.parse(). 

<!DOCTYPE html>
<html>
<body>

  <pre id="info"></pre>

  <script>
    const user = { name: "Alex", age: 21 };

    const json = JSON.stringify(user);
    const parsed = JSON.parse(json);

    const text =
      "JSON string: " + json + "\\n" +
      "Parsed name: " + parsed.name;

    document.getElementById("info").textContent = text;
  </script>

</body>
</html>

JSON keys and string values must use double quotes, and only valid JSON types like objects, arrays, numbers, strings, booleans, and null are allowed.
