JS Promises

A promise represents the result of an asynchronous operation and can be in pending, fulfilled, or rejected state.

Basic promise example

This example creates a promise that resolves after 1 second and handles the result with then() and errors with catch(). 

<!DOCTYPE html>
<html>
<body>

  <p id="status">Waiting...</p>

  <script>
    const statusEl = document.getElementById("status");

    const delay = new Promise((resolve, reject) => {
      setTimeout(() => {
        resolve("Done after 1 second");
      }, 1000);
    });

    delay
      .then(message => {
        statusEl.textContent = message;
      })
      .catch(error => {
        statusEl.textContent = "Error: " + error;
      });
  </script>

</body>
</html>

Promise chains make it easy to perform multiple async steps in order and handle errors in a single catch() at the end.
