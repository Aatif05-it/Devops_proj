JS Loops

Loops let JavaScript repeat a block of code multiple times, which is useful for lists and repetitive tasks.

for loop example

This example uses a for loop to build a list of numbers from 1 to 5 and display them in the page. 

<!DOCTYPE html>
<html>
<body>

  <ul id="numbers"></ul>

  <script>
    const list = document.getElementById("numbers");

    for (let i = 1; i <= 5; i++) {
      const li = document.createElement("li");
      li.textContent = "Number " + i;
      list.appendChild(li);
    }
  </script>

</body>
</html>

JavaScript also provides while, do...while, and for...of loops for iterating in different situations.
