JS Variables

Variables are containers for storing values so you can reuse data in your JavaScript programs.

Declaring variables

Use let or const to create variables; const is for values that should not be reassigned. 

<!DOCTYPE html>
<html>
<body>

  <p id="result"></p>

  <script>
    const firstName = "Alex";
    let age = 20;

    age = age + 1; // update the value

    const message =
      "Name: " + firstName + "\\n" +
      "Age: " + age;

    document.getElementById("result").textContent = message;
  </script>

</body>
</html>

Prefer const by default and use let when the value needs to change, avoiding the older var keyword.
