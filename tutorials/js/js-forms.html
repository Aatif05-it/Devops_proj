JS Forms & Validation

JavaScript can validate form input in the browser and prevent submission when required fields are empty or invalid.

Required fields example

This example checks that both the name and email fields are filled before allowing the form to submit. 

<!DOCTYPE html>
<html>
<body>

  <form id="signup-form">
    <label>
      Name:
      <input type="text" id="name" />
    </label><br><br>

    <label>
      Email:
      <input type="email" id="email" />
    </label><br><br>

    <button type="submit">Sign up</button>
    <p id="error" style="color: red;"></p>
  </form>

  <script>
    const form = document.getElementById("signup-form");
    const error = document.getElementById("error");

    form.addEventListener("submit", function (event) {
      error.textContent = "";

      const name = document.getElementById("name").value.trim();
      const email = document.getElementById("email").value.trim();

      if (!name || !email) {
        event.preventDefault();
        error.textContent = "Please enter both name and email.";
      }
    });
  </script>

</body>
</html>
