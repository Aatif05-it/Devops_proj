JS Objects

Objects store related data as key–value pairs and can also contain functions called methods.

Object literal

This example creates a person object with properties and uses them to build a description. 

<!DOCTYPE html>
<html>
<body>

  <p id="info"></p>

  <script>
    const person = {
      firstName: "Alex",
      lastName: "Patel",
      age: 21
    };

    const text =
      "Name: " + person.firstName + " " + person.lastName + "\\n" +
      "Age: " + person.age;

    document.getElementById("info").textContent = text;
  </script>

</body>
</html>

Object properties can be accessed with dot syntax like person.age or bracket syntax like person["age"].
