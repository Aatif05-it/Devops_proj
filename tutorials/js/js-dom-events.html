JS DOM Events

Events are actions like clicks, key presses, or page loads that JavaScript can listen for and respond to.

Click event example

This example adds a click listener to a button and changes the paragraph text when the event fires. 

<!DOCTYPE html>
<html>
<body>

  <p id="message">Click the button.</p>
  <button id="btn">Click me</button>

  <script>
    const button = document.getElementById("btn");
    const message = document.getElementById("message");

    button.addEventListener("click", function () {
      message.textContent = "Button was clicked!";
    });
  </script>

</body>
</html>

The addEventListener() method works with many event types such as click, input, and submit.
