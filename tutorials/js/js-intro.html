JavaScript Introduction

JavaScript is the programming language of the web. It runs in the browser and can update content, react to user actions, and make web pages feel interactive and alive.

Basic example

Here is a simple HTML page with some JavaScript that changes the text of a heading when a button is clicked.

<!DOCTYPE html>
<html>
<head>
  <title>My first JS page</title>
</head>
<body>

  <h1 id="title">Hello, JavaScript!</h1>
  <button onclick="changeText()">Click me</button>

  <script>
    function changeText() {
      document.getElementById("title").textContent =
        "You clicked the button!";
    }
  </script>

</body>
</html>

The code inside the <script> tag tells the browser what to do when the button is clicked, such as finding the heading and changing its text content.
