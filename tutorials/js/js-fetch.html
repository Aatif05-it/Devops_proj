JS Fetch API

The Fetch API lets JavaScript make HTTP requests and work with responses using promises.

Fetch JSON example

This example fetches JSON from a sample API and displays the title from the response. 

<!DOCTYPE html>
<html>
<body>

  <p id="result">Loading...</p>

  <script>
    const resultEl = document.getElementById("result");

    fetch("https://jsonplaceholder.typicode.com/posts/1")
      .then(response => response.json())
      .then(data => {
        resultEl.textContent = "Post title: " + data.title;
      })
      .catch(error => {
        resultEl.textContent = "Error: " + error;
      });
  </script>

</body>
</html>

A fetch call returns a promise that resolves to a Response; methods like response.json() read the response body.
