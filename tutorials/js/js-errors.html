JS Error Handling

Error handling with try...catch lets JavaScript catch runtime errors and react instead of stopping the script.

try / catch example

This example deliberately uses an undefined variable and catches the resulting error to show a friendly message. 

<!DOCTYPE html>
<html>
<body>

  <p id="info"></p>

  <script>
    try {
      const result = notDefined + 10; // ReferenceError
      document.getElementById("info").textContent = result;
    } catch (error) {
      document.getElementById("info").textContent =
        "Something went wrong: " + error.message;
    }
  </script>

</body>
</html>

Use throw to create custom errors and optionally a finally block to run cleanup code whether an error occurs or not.
