JS Conditionals

Conditional statements let JavaScript run different code depending on whether a condition is true or false.

if / else if / else

This example uses an if...else if...else chain to map a score to a letter grade. 

<!DOCTYPE html>
<html>
<body>

  <p id="grade"></p>

  <script>
    const score = 78;
    let grade;

    if (score >= 90) {
      grade = "A";
    } else if (score >= 75) {
      grade = "B";
    } else if (score >= 60) {
      grade = "C";
    } else {
      grade = "D";
    }

    document.getElementById("grade").textContent =
      "Score: " + score + " → Grade " + grade;
  </script>

</body>
</html>

Combining conditionals with logical operators like && and || makes it easy to express complex decision rules.
