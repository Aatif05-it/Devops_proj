HTML Tables

Tables display data in rows and columns using table, tr, th, and td elements.

Basic table

<table>
  <tr>
    <th>Name</th>
    <th>Role</th>
  </tr>
  <tr>
    <td>Alex</td>
    <td>Developer</td>
  </tr>
</table>

Table caption

<table>
  <caption>Student Scores</caption>
  <tr>
    <th>Student</th>
    <th>Score</th>
  </tr>
  <tr>
    <td>Sam</td>
    <td>95</td>
  </tr>
</table>
