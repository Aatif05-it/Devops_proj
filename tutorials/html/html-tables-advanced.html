HTML Tables Advanced

Advanced table features include table headers, grouped sections, and cells that span multiple rows or columns. These features make data tables easier to read and more accessible.

Head, body, and foot sections

Use <thead>, <tbody>, and <tfoot> to group header rows, body rows, and footer rows inside a table. 

<table>
  <thead>
    <tr>
      <th>Month</th>
      <th>Sales</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>January</td>
      <td>$1,200</td>
    </tr>
    <tr>
      <td>February</td>
      <td>$950</td>
    </tr>
  </tbody>
  <tfoot>
    <tr>
      <th>Total</th>
      <th>$2,150</th>
    </tr>
  </tfoot>
</table>

Grouping rows like this makes tables easier to style and helps assistive technologies understand the structure.

Rowspan and colspan

The rowspan attribute lets a cell span multiple rows, and colspan lets a cell span multiple columns. 

<table>
  <tr>
    <th rowspan="2">Name</th>
    <th colspan="2">Score</th>
  </tr>
  <tr>
    <th>Math</th>
    <th>Science</th>
  </tr>
  <tr>
    <td>Alex</td>
    <td>90</td>
    <td>85</td>
  </tr>
</table>

Spanning cells is useful for grouped headings or summarizing data across several rows or columns.

Scope for accessible headers

Add the scope attribute to header cells so screen readers know which row or column a header belongs to. 

<table>
  <tr>
    <th scope="col">Name</th>
    <th scope="col">Email</th>
  </tr>
  <tr>
    <th scope="row">Alex</th>
    <td>alex@example.com</td>
  </tr>
  <tr>
    <th scope="row">Sam</th>
    <td>sam@example.com</td>
  </tr>
</table>

scope="col" marks a header as describing the column below, and scope="row" describes the row to the right, improving accessibility for data tables.
