HTML Basic

In this chapter you learn basic HTML building blocks: headings, paragraphs, and links.

Headings

HTML headings go from h1 (largest) to h6 (smallest) and give structure to your page.

<h1>Main title</h1>
<h2>Section heading</h2>
<h3>Sub heading</h3>

Paragraphs

Use the p element to add blocks of text.

<p>This is a paragraph of text on a web page.</p>

Links

The a element with an href attribute creates a hyperlink to another page.

<a href="https://example.com">Visit Example</a>
