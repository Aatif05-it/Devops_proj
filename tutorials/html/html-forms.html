HTML Forms

Forms are used to collect user input and send it to a server.

<form>
  <label>Name:
    <input type="text" name="username">
  </label>
  <br>
  <label>Email:
    <input type="email" name="email">
  </label>
  <br>
  <button type="submit">Submit</button>
</form>

Inputs, labels, and buttons are common form controls used to build login forms, contact forms, and more.
