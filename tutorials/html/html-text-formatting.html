HTML Text Formatting

Text formatting elements let you mark important words, add emphasis, show code, and display small notes while keeping the meaning of the text clear.

Strong importance vs emphasis

Use <strong> for very important text and <em> for stressed or emphasized words. Browsers usually render strong as bold and em as italic. 

<p>Please <strong>save</strong> your work.</p>
<p>This is <em>really</em> important.</p>

Highlighting and small text

<mark> highlights part of a sentence, and <small> is good for fine print or side notes. 

<p>Read the <mark>updated</mark> rules.</p>
<p><small>*Terms and conditions apply.</small></p>

Superscript and subscript

Use <sub> for text below the baseline and <sup> for text above the baseline. They are common in formulas and scientific notation. 

<p>Water is H<sub>2</sub>O.</p>
<p>The area is 5m<sup>2</sup>.</p>

Code and keyboard input

The <code> element marks inline code, and <kbd> shows keys the user should press. 

<p>Use <code>console.log()</code> to print to the console.</p>
<p>Press <kbd>Ctrl</kbd> + <kbd>S</kbd> to save.</p>
