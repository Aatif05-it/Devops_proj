HTML Links

Links use the a element and the href attribute to navigate to other pages and resources.

Basic link

<a href="https://example.com">Visit Example</a>

Open in a new tab

<a href="https://example.com" target="_blank" rel="noreferrer">
  Open in new tab
</a>

Link to a section on the same page

<a href="#top">Back to top</a>

<h2 id="top">Top of the page</h2>

Using href="#id" jumps to the element that has the matching id attribute.
