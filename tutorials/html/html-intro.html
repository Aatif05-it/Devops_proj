HTML Introduction

HTML is the standard markup language for creating web pages.

Basic example

<!DOCTYPE html>
<html>
<head>
  <title>My first page</title>
</head>
<body>

  <h1>Hello, world!</h1>
  <p>This is my first HTML page.</p>

</body>
</html>

This code defines the structure of a basic web page: a head section with a title and a body with content.
