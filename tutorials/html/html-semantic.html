Semantic HTML

Semantic elements describe their meaning, making pages more accessible and easier to maintain.

Common semantic elements

<header>Site header</header>
<nav>Main navigation</nav>
<main>
  <section>A section of content</section>
  <article>An independent article</article>
</main>
<footer>Site footer</footer>

Using these instead of generic div elements improves structure for screen readers and search engines.
