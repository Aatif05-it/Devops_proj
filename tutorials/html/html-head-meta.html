HTML Head & Meta

The head section of an HTML document contains information about the page: title, character encoding, responsive settings, SEO meta tags, and links to CSS or icons.

Basic head structure

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>My Page</title>
</head>
<body>
  ...
</body>
</html>

The lang attribute helps browsers and screen readers, while meta charset="UTF-8" ensures most characters are displayed correctly.

Viewport meta (responsive)

The viewport tag makes your layout scale correctly on phones and tablets. 

<meta name="viewport" content="width=device-width, initial-scale=1">

width=device-width sets the page width to the device width and initial-scale=1 sets the initial zoom level.

Description and keywords

Search engines often use the description meta in search results. 

<meta name="description"
      content="Learn HTML, CSS, and JavaScript with simple examples.">
<meta name="keywords"
      content="HTML, CSS, JavaScript, tutorial">

Description should be a short summary of the page. Keywords are less important today but still used in some tools.

Linking CSS and favicon

<link rel="stylesheet" href="styles.css">
<link rel="icon" type="image/png" href="/assets/images/favicon.png">

Use link rel="stylesheet" to connect your CSS files and rel="icon" to set the small icon shown in the browser tab.
