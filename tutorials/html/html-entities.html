HTML Entities

HTML entities are special codes that display characters which are reserved in HTML or difficult to type on a keyboard.

Why entities are needed

Characters like < and > are used to mark tags, so when you want to show them as text you must use an entity. 

<p>This will show &amp; and &lt; and &gt; on the page.</p>

Here &amp; shows &, &lt; shows <, and &gt; shows >.

Common HTML entities

Some useful entities for symbols and currency:

<p>Copyright: &copy; 2025 CodeNexa Academy</p>
<p>Registered trademark: &reg;</p>
<p>Euro price: &euro; 10</p>
