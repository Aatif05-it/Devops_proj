HTML Form File Upload

File uploads in HTML are handled with the <input type="file"> element inside a form, usually with the form encoded as multipart/form-data.

Basic file upload field

To let a user choose a single file, add an input with type="file" and make sure the form has enctype="multipart/form-data". 

<form action="/upload" method="post" enctype="multipart/form-data">
  <label for="resume">Upload your resume:</label>
  <input type="file" id="resume" name="resume">

  <button type="submit">Upload</button>
</form>

Without the correct enctype, the browser will not send the file data properly to the server.

Allowing multiple files

Add the boolean multiple attribute to let users select more than one file from the file picker dialog. 

<form action="/upload-photos" method="post" enctype="multipart/form-data">
  <label for="photos">Select photos:</label>
  <input
    type="file"
    id="photos"
    name="photos"
    multiple>

  <button type="submit">Upload Photos</button>
</form>

The server will receive a collection of files under the same field name and can loop over them when processing the upload.

Restricting allowed file types

Use the accept attribute to hint which file types the user can select, such as only images or PDFs. 

<form action="/upload-documents" method="post" enctype="multipart/form-data">
  <label for="docs">Upload documents (PDF only):</label>
  <input
    type="file"
    id="docs"
    name="documents"
    accept=".pdf">

  <button type="submit">Upload Documents</button>
</form>

Remember that accept is only a client-side filter, so the server must still validate all uploaded files securely.

Adding simple client-side preview

With JavaScript and the File API, you can show a preview of selected files before submitting the form. 

<form>
  <label for="avatar">Choose a profile picture:</label>
  <input type="file" id="avatar" name="avatar" accept="image/*">

  <div id="preview"></div>
</form>

<script>
  const input = document.getElementById('avatar');
  const preview = document.getElementById('preview');

  input.addEventListener('change', () => {
    preview.innerHTML = '';

    const file = input.files[0];
    if (!file) return;

    const img = document.createElement('img');
    img.src = URL.createObjectURL(file);
    img.alt = file.name;
    img.style.maxWidth = '200px';

    preview.appendChild(img);
  });
</script>

This makes the upload experience clearer for users, especially when selecting profile photos or gallery images.
