HTML Attributes

Attributes provide extra information about an element, such as id, class, style, or a link target.[web:124][web:127]

Basic syntax

<tagname attribute="value">Content</tagname>

Attributes are always written in the opening tag as name="value" pairs.[web:128][web:134]

Common attributes: id and class

<p id="intro" class="lead-text">
  This paragraph has an id and a class.
</p>

The id attribute uniquely identifies one element, while class can be shared by many elements and is heavily used with CSS.[web:127][web:143]

The href attribute on links

<a href="https://example.com">Visit Example</a>

Here href is the attribute name and the URL is the value, telling the browser where the link should go.

The src attribute on images

<img src="photo.jpg" alt="A landscape photo" width="300">

The src attribute defines the image file, alt provides alternative text, and width sets the display size in pixels.

Boolean attributes

<input type="text" required>
<button disabled>You cannot click me</button>

Boolean attributes like required and disabled can appear without a value and turn on special behavior for the element.
