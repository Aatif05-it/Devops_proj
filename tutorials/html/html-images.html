HTML Images

The img element embeds an image using the src attribute and should always include alternative text.

Basic image

<img src="photo.jpg" alt="A landscape photo">

Setting width and height

<img src="logo.png" alt="Site logo" width="120" height="120">

Width and height can be given in pixels or controlled with CSS for responsive layouts.

Using images as links

<a href="index.html">
  <img src="logo.png" alt="Back to home">
</a>
