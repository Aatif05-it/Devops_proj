HTML Iframe & Embed

Iframes let you display another web page inside your page. They are often used for maps, videos, or external tools.

Basic iframe

The <iframe> element takes a src URL and optional width and height attributes. 

<iframe
  src="https://example.com"
  width="600"
  height="400"
  title="Example website">
</iframe>

Always set a descriptive title for accessibility so screen readers understand what the iframe contains.

Responsive iframe (full width)

For responsive layouts you can use CSS to make the iframe stretch to 100% width and control the height with a ratio. 

<div class="iframe-wrapper">
  <iframe
    src="https://www.example.com"
    title="Responsive demo"
    frameborder="0"
    allowfullscreen>
  </iframe>
</div>

Security considerations

Use the sandbox attribute to limit what the embedded page can do (for example, block scripts or form submission) when you do not fully trust it. 

<iframe
  src="https://untrusted.example"
  sandbox="allow-same-origin allow-forms"
  title="Sandboxed content">
</iframe>
