HTML Paragraphs

The p element defines a block of text and automatically adds some spacing before and after.

Basic paragraph

<p>This is a paragraph of text on a web page.</p>

Line breaks with br

<p>First line<br>Second line</p>

The br element forces a line break inside a paragraph without starting a new paragraph.

Preformatted text

<pre>
Line 1
  Line 2 keeps spaces
Line 3
</pre>

The pre element keeps spaces and line breaks exactly as written, useful for code blocks or plain text output.
