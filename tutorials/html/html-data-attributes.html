HTML Data Attributes

Data attributes let you store custom information on HTML elements using attributes that start with data-. They are often used by JavaScript and CSS.

Basic syntax

A data attribute has a name and value: data-name="value". You can define as many as you need on any HTML element. 

<button
  data-course-id="html101"
  data-level="beginner">
  Start HTML course
</button>

Here data-course-id and data-level store extra information that does not affect how the button is rendered.

Using data attributes with JavaScript

In JavaScript you can read data attributes through the dataset property, which exposes them as camelâ€‘cased keys. 

<button id="startBtn"
        data-course-id="html101"
        data-level="beginner">
  Start HTML course
</button>

<script>
  const btn = document.getElementById("startBtn");
  const courseId = btn.dataset.courseId; // "html101"
  const level = btn.dataset.level;       // "beginner";

  console.log(courseId, level);
</script>

This pattern is useful for attaching IDs, types, or configuration options to elements without hardâ€‘coding them in your script.

Data attributes for styling hooks

CSS can also select elements based on data attributes using attribute selectors, which is handy for stateâ€‘based styling. 

<button data-state="active">Active tab</button>
<button data-state="inactive">Inactive tab</button>

<style>
  button[data-state="active"] {
    background: #04aa6d;
    color: #fff;
  }
</style>

JavaScript can update the data attribute value and CSS will automatically reflect the change in the UI.
