CSS Borders & Outlines

Borders appear inside the elementâ€™s box, while outlines sit outside and do not affect layout.

Basic borders

.card {
  border: 1px solid #d0d3d6;
  border-radius: 8px;
}

The border shorthand sets width, style, and color. Use border-radius for rounded corners.

Individual sides

.underline {
  border-bottom: 2px solid #04aa6d;
}

You can style individual sides with border-top, border-right, border-bottom, and border-left.

Outlines for focus

button:focus {
  outline: 3px solid #2563eb;
  outline-offset: 2px;
}

Outlines are often used to show keyboard focus. Avoid removing focus outlines without providing an accessible replacement.
