CSS Typography

Typography in CSS controls fonts, sizes, line spacing, and alignment to make text readable and visually pleasing.

Setting the font

body {
  font-family: "InterLocal", system-ui, -apple-system, BlinkMacSystemFont,
    "Segoe UI", sans-serif;
}

font-family lists preferred fonts in order. The browser uses the first available one.

Font size, weight, and style

h1 {
  font-size: 2rem;
  font-weight: 600;
}

em {
  font-style: italic;
}

Use font-size (often with rem units) for scalable text and font-weight for boldness.

Line height and letter spacing

p {
  line-height: 1.6;
  letter-spacing: 0.01em;
}

line-height improves readability by adding vertical space between lines; letter-spacing adjusts spacing between characters.

Text alignment and decoration

.centered {
  text-align: center;
}

a {
  text-decoration: none;
}

text-align controls horizontal alignment inside a block and text-decoration is often used to remove underlines from links.
