CSS Backgrounds

Background properties let you control the color, images, and positioning behind an elementâ€™s content.

Background color

.card {
  background-color: #ffffff;
}

background-color sets a solid color behind the elementâ€™s content and padding.

Background image

body {
  background-image: url("images/pattern.png");
}

background-image uses the given image file as the background. By default it repeats to fill the element.

Background repeat, position, and size

.hero {
  background-image: url("images/hero.jpg");
  background-repeat: no-repeat;
  background-position: center;
  background-size: cover;
}

no-repeat prevents tiling, center positions the image, and cover scales it to fill the area while keeping its aspect ratio.
