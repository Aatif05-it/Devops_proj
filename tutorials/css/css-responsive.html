CSS Responsive Design

Responsive design makes your layout adapt to different screen sizes, from mobile phones to large desktops.

Using media queries

.card-grid {
  display: grid;
  grid-template-columns: 1fr;
}

/* Screens 768px and wider */
@media (min-width: 768px) {
  .card-grid {
    grid-template-columns: 1fr 1fr;
  }
}

The @media rule applies different CSS when the viewport meets a condition, such as a minimum width.

Flexible images

img {
  max-width: 100%;
  height: auto;
}

This keeps images from overflowing their container while maintaining their aspect ratio.
