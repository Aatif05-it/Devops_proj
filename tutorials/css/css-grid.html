CSS Grid (Intro)

CSS Grid is a twoâ€‘dimensional layout system that lets you create rows and columns for more complex page layouts.

Basic grid container

.grid {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr;
  gap: 12px;
}

display: grid; turns an element into a grid container. grid-template-columns defines three equalâ€‘width columns, and gap sets spacing between grid items.

Fixed and flexible tracks

.grid-layout {
  display: grid;
  grid-template-columns: 200px 1fr;
}

This creates a first column that is always 200px wide and a second column that takes up the remaining space.

Placing items with column and row

.item-wide {
  grid-column: 1 / 3; /* spans first two columns */
}

grid-column and grid-row let you make items span across multiple tracks to create more interesting layouts.
