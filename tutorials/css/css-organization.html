CSS Naming & Organization

Organizing CSS with a naming convention makes large projects easier to read and maintain.

BEM-style class names

.card { ... }               /* Block */
.card__title { ... }        /* Element */
.card--featured { ... }     /* Modifier */

BEM (Block Element Modifier) uses double underscores and double hyphens to show component structure.

File structure

assets/css/
  base.css      /* reset, typography */
  layout.css    /* grid, header, footer */
  components.css/* cards, buttons, forms */
  pages.css     /* page-specific tweaks */

Splitting styles into base, layout, components, and pages keeps your CSS scalable as the site grows.
