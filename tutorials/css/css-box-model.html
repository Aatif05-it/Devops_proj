CSS Box Model

In CSS, every element is treated as a rectangular box made up of content, padding, border, and margin.

Box model layers

.box {
  margin: 16px;      /* space outside the border */
  border: 2px solid #04aa6d;
  padding: 12px;     /* space between border and content */
  width: 200px;
}

The total visual width of the element is its content width plus padding, border, and left/right margins.

Content-box vs border-box

/* Default: content-box */
.box-default {
  width: 200px;
}

/* border-box: width includes padding and border */
.box-border-box {
  box-sizing: border-box;
  width: 200px;
  padding: 16px;
  border: 4px solid #04aa6d;
}

With box-sizing: border-box;, the declared width includes padding and border, which makes layouts easier to control.
