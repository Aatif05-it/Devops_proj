CSS Flexbox Deep Dive

Flexbox is a oneâ€‘dimensional layout system ideal for rows or columns of items that need to align and distribute space.

Axis and alignment

.nav {
  display: flex;
  align-items: center;        /* cross axis */
  justify-content: space-between; /* main axis */
}

justify-content works along the main axis; align-items works along the cross axis.

Wrapping and gaps

.tag-list {
  display: flex;
  flex-wrap: wrap;
  gap: 0.5rem;
}

flex-wrap allows items to move to new lines, and gap adds consistent spacing between them.
