CSS Overflow

The overflow property controls what happens when content is too large for its container.

Common overflow values

.box-visible {
  overflow: visible;  /* default: content spills out */
}

.box-hidden {
  overflow: hidden;   /* extra content is clipped */
}

.box-scroll {
  overflow: scroll;   /* scrollbars always shown */
}

.box-auto {
  overflow: auto;     /* scrollbars only if needed */
}

Use hidden to clip content, scroll for persistent scrollbars, or auto to show them only when content overflows.

Axis-specific overflow

.box-y-scroll {
  overflow-x: hidden;
  overflow-y: scroll;
}

overflow-x and overflow-y let you control horizontal and vertical scrolling separately.
