CSS Display & Visibility

The display and visibility properties control how elements participate in layout and whether they are visible.

Common display values

.block {
  display: block;       /* starts on new line, full width */
}

.inline {
  display: inline;      /* flows inside a line */
}

.inline-block {
  display: inline-block; /* inline, but respects width/height */
}

Block elements stack vertically; inline elements sit in text flow; inlineâ€‘block combines both behaviors.

Hiding elements

.hidden {
  display: none;      /* removed from layout */
}

.invisible {
  visibility: hidden; /* kept in layout, but not visible */
}

display: none removes the element from layout. With visibility: hidden, the space remains reserved.
