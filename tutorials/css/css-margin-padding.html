CSS Margin & Padding

Margin adds space outside the border; padding adds space inside the border around the content.

Basic spacing

.card {
  margin: 16px;
  padding: 12px;
}

This gives each card 16px of space from surrounding elements and 12px of inner space around its content.

Shorthand values

.box {
  margin: 10px 20px;        /* vertical | horizontal */
  padding: 8px 12px 16px 4px; /* top right bottom left */
}

Two values apply to vertical and horizontal axes. Four values apply to each side in clockwise order.
