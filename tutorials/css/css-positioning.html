CSS Positioning

The position property controls how elements are placed on the page relative to the normal document flow.

Relative and absolute

.relative-box {
  position: relative;
}

.absolute-child {
  position: absolute;
  top: 10px;
  right: 10px;
}

An absolutely positioned element is taken out of flow and positioned relative to the nearest ancestor with position: relative;.

Fixed and sticky

.fixed-header {
  position: fixed;
  top: 0;
  left: 0;
  right: 0;
}

.sticky-note {
  position: sticky;
  top: 1rem;
}

Fixed elements stay in place relative to the viewport. Sticky elements scroll normally until they reach a threshold, then stick.
