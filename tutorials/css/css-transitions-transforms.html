CSS Transitions & Transforms

Transforms change how elements look or where they sit; transitions make those changes animate smoothly over time.

Basic transition

.btn-animate {
  background-color: #04aa6d;
  transition: background-color 0.2s ease-in-out;
}

.btn-animate:hover {
  background-color: #028557;
}

A transition needs at least a property and a duration, and can also define timing function and delay.

Using transforms

.card-hover {
  transition: transform 0.25s ease;
}

.card-hover:hover {
  transform: translateY(-4px) scale(1.02);
}

The transform property can move, scale, rotate, or skew elements, often combined with transitions for smooth hover effects.
