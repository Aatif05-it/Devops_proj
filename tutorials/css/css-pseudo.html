CSS Pseudo-classes & Elements

Pseudo-classes target elements in a certain state, while pseudo-elements style specific parts of an element.

Common pseudo-classes

a:link    { color: #2563eb; }
a:visited { color: #7c3aed; }
a:hover   { text-decoration: underline; }
a:active  { color: #b91c1c; }

input:focus {
  outline: 2px solid #2563eb;
}

Pseudo-classes like :hover, :focus, and :active respond to user interaction and element state.

Pseudo-elements

.title::before {
  content: "â—† ";
  color: #f97316;
}

p::first-line {
  font-weight: 600;
}

Pseudo-elements such as ::before and ::after let you inject and style extra decorative content without extra HTML.
