CSS Syntax

CSS rules are written as selectors followed by a set of property–value pairs inside curly braces.

Basic rule structure

selector {
  property: value;
}

The selector chooses which HTML elements to style, and each declaration sets one visual property for those elements.

Example: style all paragraphs

p {
  color: #333;
  font-size: 16px;
  line-height: 1.6;
}

This rule makes all <p> elements dark gray, sets the font size to 16px, and increases line spacing for better readability.

Multiple selectors and comments

/* Headings share the same color */
h1, h2, h3 {
  color: #04aa6d;
}

/* Single-line comment */
a {
  text-decoration: none;
}

You can target several selectors at once by separating them with commas, and you can add comments using /* ... */.
