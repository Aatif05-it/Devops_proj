CSS Links & Buttons

Links and buttons are key interactive elements. Good styling makes them clear and easy to use.

Basic link styling

a {
  color: #04aa6d;
  text-decoration: none;
}

a:hover {
  text-decoration: underline;
}

Button-like links

.btn-link {
  display: inline-block;
  padding: 0.55rem 1.15rem;
  border-radius: 999px;
  background-color: #04aa6d;
  color: #ffffff;
}

Turning a link into an inlineâ€‘block and adding padding, background, and border radius makes it look like a button.

Styling <button> elements

button.primary {
  border: none;
  border-radius: 999px;
  padding: 0.55rem 1.15rem;
  background-color: #04aa6d;
  color: #ffffff;
  cursor: pointer;
}

Use cursor: pointer; on interactive controls so users get a clear visual cue.
