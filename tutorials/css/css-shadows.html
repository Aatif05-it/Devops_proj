CSS Shadows

Shadows add depth and separation using box-shadow for elements and text-shadow for text.

Box shadows

.card {
  box-shadow: 0 10px 25px rgba(15, 23, 42, 0.12);
  border-radius: 12px;
}

The values are horizontal offset, vertical offset, blur radius, and color, plus optional spread.

Soft text shadow

h1.glow {
  text-shadow: 0 2px 6px rgba(15, 23, 42, 0.35);
}

Subtle text shadows improve contrast on images but should not reduce readability. [web:89]
