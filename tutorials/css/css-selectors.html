CSS Selectors

Selectors decide which HTML elements a CSS rule applies to. Different selector types let you target elements in flexible ways.

Type selector

Targets all elements with a given tag name.

p {
  color: #333;
}

This rule styles every <p> element on the page.

Class selector

Targets elements that share the same class attribute.

.highlight {
  background-color: yellow;
}

Any element with class="highlight" gets a yellow background.

ID selector

Targets a single unique element with a specific id.

#main-title {
  font-size: 2rem;
}

Only the element with id="main-title" is affected by this rule.

Descendant selector

Targets elements that are inside other elements, matching a structure. 

nav a {
  color: #04aa6d;
}

This styles all <a> links that live inside a <nav> element.
