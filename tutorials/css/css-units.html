CSS Units

CSS supports absolute units like px and relative units like em, rem, %, vw, and vh. Choosing the right unit is important for responsive design.

Pixels (px)

h1 {
  font-size: 32px;
}

.box {
  width: 200px;
  height: 100px;
}

Pixels are fixed-size units; they do not scale automatically with the userâ€™s default font size.

Relative font units: em and rem

body {
  font-size: 16px;
}

p {
  font-size: 1rem;   /* 16px */
}

p.small {
  font-size: 0.875rem; /* 14px */
}

button {
  font-size: 1em;    /* relative to button's parent font size */
}

rem is based on the root font size (html), while em is based on the current elementâ€™s parent. Using rem helps keep sizes more predictable.

Percent and viewport units

.half-width {
  width: 50%;
}

.full-screen {
  width: 100vw;  /* viewport width */
  height: 100vh; /* viewport height */
}

Percentage units are relative to the parent box. Viewport units (vw, vh) are relative to the size of the browser window.
