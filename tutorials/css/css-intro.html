CSS Introduction

CSS (Cascading Style Sheets) is the language used to style and lay out web pages. HTML provides the structure; CSS controls colors, fonts, spacing, and layout.

Basic example

Here is a simple HTML page with some CSS that changes the font and the color of a heading.

<!DOCTYPE html>
<html>
<head>
  <title>My first styled page</title>
  <style>
    body {
      font-family: Arial, sans-serif;
      background-color: #f5f5f5;
    }

    h1 {
      color: #04aa6d;
      text-align: center;
    }

    p {
      color: #333;
    }
  </style>
</head>
<body>

  <h1>Hello, CSS!</h1>
  <p>This page is styled with a simple CSS rule set.</p>

</body>
</html>

The rules inside the <style> tag tell the browser how to display each element, such as setting the background color of the page and the color of the heading.
