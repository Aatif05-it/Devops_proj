CSS Colors

CSS lets you set colors for text, backgrounds, borders, and more using keywords, hexadecimal values, RGB, and HSL.

Named colors

h1 {
  color: blue;
}

p.warning {
  color: red;
}

CSS has many predefined color names such as red, blue, green, and orange.

Hex colors

body {
  background-color: #f5f5f5;
}

button.primary {
  background-color: #04aa6d;
  color: #ffffff;
}

Hex values start with # followed by 3 or 6 hexadecimal digits, such as #04aa6d.

RGB and RGBA

.box {
  background-color: rgb(255, 200, 0);
}

.box-overlay {
  background-color: rgba(0, 0, 0, 0.5);
}

rgb() uses red, green, and blue values from 0 to 255, while rgba() adds an alpha channel for transparency.
