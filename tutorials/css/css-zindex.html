CSS Z-Index & Stacking

When elements overlap, the z-index property controls which one appears on top along the zâ€‘axis.

Basic z-index

.box-a {
  position: relative;
  z-index: 1;
}

.box-b {
  position: relative;
  z-index: 2; /* on top of .box-a */
}

z-index only works on positioned elements (position not static). Higher values sit above lower values.

Stacking context

.parent {
  position: relative;
  z-index: 1; /* creates a stacking context */
}

.child {
  position: absolute;
  z-index: 999; /* still limited to its parent context */
}

A stacking context is like a local 3D layer group: children are stacked relative to the parent, not the whole page.
