SQL CASE Expression

The CASE expression lets a query apply IFâ€‘THENâ€‘ELSE logic per row, returning different values depending on conditions.

CASE can appear in SELECT, ORDER BY, and UPDATE, and supports multiple WHEN branches plus an optional ELSE.

Simple CASE in SELECT

This example converts numeric scores into letter grades directly in the output. 

SELECT
  StudentName,
  Score,
  CASE
    WHEN Score >= 90 THEN 'A'
    WHEN Score >= 80 THEN 'B'
    WHEN Score >= 70 THEN 'C'
    ELSE 'F'
  END AS Grade
FROM Students;

The first true WHEN condition determines the returned value; if none match, the ELSE branch is used.

CASE in ORDER BY

CASE can customize sort order, for example pushing NULLs to the bottom or prioritizing VIP customers. 

SELECT CustomerName, City, Country
FROM Customers
ORDER BY
  CASE
    WHEN City IS NULL THEN 1
    ELSE 0
  END,
  City;

Here rows with a missing city sort after those with a known city, while still sorting alphabetically within each group.
