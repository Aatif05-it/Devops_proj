SQL Create & Drop DB

SQL can create new databases with CREATE DATABASE and remove them with DROP DATABASE, operations usually restricted to administrators.

Dropping a database permanently deletes all its tables and data, so it should only be done on development or with full backups.

Create a database

This example creates a database named ShopDB and then switches to it using USE. 

CREATE DATABASE ShopDB;

-- Select the new database (syntax may vary)
USE ShopDB;

Some systems support options such as character set and collation when you create a database to control how text is stored and compared.

Drop a database

The following command removes a database completely; many DBMSs support IF EXISTS to avoid errors when the database is missing. 

DROP DATABASE IF EXISTS ShopDB;
