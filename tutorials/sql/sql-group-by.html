SQL GROUP BY

The GROUP BY clause groups rows that share the same values in one or more columns so aggregate functions can be applied per group.

It is widely used for summaries like â€œorders per customerâ€, â€œsales per monthâ€, or â€œnumber of users per countryâ€.

Count customers per country

This example counts how many customers exist in each country. 

SELECT Country,
       COUNT(*) AS CustomerCount
FROM Customers
GROUP BY Country;

Every distinct country value becomes a group, and COUNT(*) returns the number of rows in each group.

Group by multiple columns

You can group by several columns to get counts or sums for each unique combination. 

SELECT Country,
       City,
       COUNT(*) AS CustomerCount
FROM Customers
GROUP BY Country, City
ORDER BY Country, City;

Grouping by both country and city distinguishes cities with the same name in different countries, giving more granular statistics.
