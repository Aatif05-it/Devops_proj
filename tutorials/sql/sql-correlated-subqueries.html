Correlated EXISTS subquery

EXISTS with a correlated subquery is often used to test whether a related row exists without caring about its exact values. 

SELECT CustomerID, CustomerName
FROM Customers AS c
WHERE EXISTS (
  SELECT 1
  FROM Orders AS o
  WHERE o.CustomerID = c.CustomerID
);

This query returns only customers who have at least one matching order, similar to an inner join but expressed with EXISTS.
