SQL Aggregate Functions

Aggregate functions calculate a single value from many rows, such as counting rows or summing amounts.

Common aggregate functions include COUNT(), SUM(), AVG(), MIN(), and MAX(), often used in reports and dashboards.

Basic aggregate examples

This example counts customers and calculates the minimum, maximum, and average order totals. 

SELECT
  COUNT(*)        AS TotalCustomers,
  MIN(TotalAmount) AS SmallestOrder,
  MAX(TotalAmount) AS LargestOrder,
  AVG(TotalAmount) AS AverageOrder
FROM Orders;

Aggregates ignore NULL values by default, so only rows with real numeric values contribute to these calculations.

SUM for totals

SUM() is commonly used to compute totals like revenue, quantities, or durations over a set of rows. 

SELECT SUM(TotalAmount) AS TotalRevenue
FROM Orders
WHERE OrderDate >= '2025-01-01';

Combining aggregates with filters allows you to compute metrics for specific time periods, regions, or customer segments.
