SQL User-Defined Functions

User-defined functions (UDFs) return computed values based on input parameters, usable anywhere built-in functions appear in queries.

Scalar UDFs return single values; table-valued UDFs return result sets for use in FROM clauses.

Scalar function example

This function calculates tax based on amount and rate, reusable across queries. 

CREATE FUNCTION CalculateTax(
  amount DECIMAL(10,2),
  rate DECIMAL(4,2)
)
RETURNS DECIMAL(10,2)
DETERMINISTIC
RETURN amount * (rate / 100);

Use in SELECT like any function: CalculateTax(1000, 18.5) returns 185.00.

Using the function

Functions integrate naturally into expressions, joins, and aggregations. 

SELECT
  ProductName,
  Price,
  CalculateTax(Price, 18.5) AS TaxAmount,
  Price + CalculateTax(Price, 18.5) AS Total
FROM Products;

Marking functions DETERMINISTIC helps the optimizer cache results and improve query plans.
