SQL String Functions

String functions manipulate text values for tasks like trimming spaces, changing case, measuring length, and extracting substrings.

Common functions include LOWER(), UPPER(), SUBSTRING(), TRIM(), LENGTH(), and CONCAT().

Cleaning and formatting text

This example normalizes email text and formats names consistently. 

SELECT
  TRIM(LOWER(Email)) AS NormalizedEmail,
  CONCAT(
    UPPER(LEFT(FirstName, 1)),
    LOWER(SUBSTRING(FirstName, 2))
  ) AS ProperFirstName
FROM Users;

Chaining functions allows complex transformations in SQL, which is useful when preparing data for analytics or exports.

Extracting parts of strings

Functions like SUBSTRING() and POSITION() (or INSTR()) help extract or locate pieces of text. 

-- Area code from '(022) 555-1234'
SELECT
  SUBSTRING(Phone, 2, 3) AS AreaCode
FROM Customers;

-- Position of '@' in an email
SELECT
  POSITION('@' IN Email) AS AtIndex
FROM Users;
