SQL HAVING

The HAVING clause filters groups created by GROUP BY, similar to how WHERE filters individual rows.

Unlike WHERE, HAVING can use aggregate functions such as COUNT() or SUM() in its conditions.

Filter groups by count

This query returns only countries that have more than 5 customers. 

SELECT Country,
       COUNT(*) AS CustomerCount
FROM Customers
GROUP BY Country
HAVING COUNT(*) > 5;

The condition in HAVING runs after grouping, so COUNT(*) already represents the number of customers in each country.

SELECT CustomerID,
       SUM(TotalAmount) AS TotalSpent
FROM Orders
GROUP BY CustomerID
HAVING SUM(TotalAmount) > 1000
ORDER BY TotalSpent DESC;
