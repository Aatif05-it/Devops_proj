SQL Window Functions

Window functions compute values across a set of rows related to the current row, without collapsing groups like aggregate functions do.

Use ROW_NUMBER(), RANK(), LAG(), and FIRST_VALUE() with OVER() to analyze trends and rankings.

Ranking with ROW_NUMBER

This ranks customers by total sales within each city. 

SELECT
  City,
  CustomerName,
  TotalSales,
  ROW_NUMBER() OVER (
    PARTITION BY City 
    ORDER BY TotalSales DESC
  ) AS SalesRank
FROM CustomerSales;

PARTITION BY divides rows into groups; ORDER BY determines ranking within each partition.

Running totals and LAG

Window functions calculate cumulative values and compare with previous rows. 

SELECT
  OrderDate,
  DailySales,
  SUM(DailySales) OVER (ORDER BY OrderDate) AS RunningTotal,
  LAG(DailySales) OVER (ORDER BY OrderDate) AS PreviousDay
FROM DailySales;
