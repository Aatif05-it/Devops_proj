SQL ORDER BY

The ORDER BY clause sorts query results by one or more columns in ascending or descending order. [web:347][web:322]

Ascending order is the default, while the DESC keyword sorts values from highest to lowest or Z to A. [web:325][web:354]

Sort by one column

This query returns all customers sorted alphabetically by name. [web:347][web:313] 

SELECT CustomerID, CustomerName, City
FROM Customers
ORDER BY CustomerName ASC;

Adding ASC is optional because most SQL dialects use ascending order by default when no direction is specified. [web:347][web:325]

Sort by multiple columns

SELECT CustomerName, Country, City
FROM Customers
ORDER BY Country ASC,
         City DESC;

Here customers are grouped by country in alphabetical order, and within each country cities are listed from highest to lowest. [web:316][web:313]
