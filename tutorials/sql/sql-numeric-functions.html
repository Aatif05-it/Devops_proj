SQL Numeric Functions

Numeric functions perform arithmetic and rounding, which is useful when calculating currency, percentages, or bucketed values.

Common examples include ROUND(), CEILING(), FLOOR(), ABS(), and sometimes POWER() and MOD().

Rounding values

This query rounds average order amounts to two decimal places. 

SELECT
  CustomerID,
  ROUND(AVG(TotalAmount), 2) AS AvgOrderRounded
FROM Orders
GROUP BY CustomerID;

Rounding inside the query ensures consistent formatting and avoids floatingâ€‘point noise in reports.

Ceiling, floor, and absolute

CEILING() and FLOOR() move numbers up or down to the nearest integer, while ABS() removes the sign. 

SELECT
  Price,
  CEILING(Price) AS RoundedUp,
  FLOOR(Price)   AS RoundedDown,
  ABS(Discount)  AS DiscountMagnitude
FROM Products;

These functions help with bucketing values, cleaning negative measurements, and preparing data for charts.
