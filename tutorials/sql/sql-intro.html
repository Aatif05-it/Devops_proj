SQL Introduction

SQL (Structured Query Language) is a standard language used to store, query, and manage data in relational databases such as MySQL, PostgreSQL, and SQL Server.

SQL works with tables made of rows and columns, and supports commands to select, insert, update, and delete data as well as create and modify database objects.

Simple SELECT example

This query selects all columns from a table named Customers, returning one row for each customer.

SELECT *
FROM Customers;

The * symbol means "all columns"; in later lessons you will usually select only the specific columns you need for better performance and clarity.
