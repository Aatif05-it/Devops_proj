SQL SELECT DISTINCT

The SELECT DISTINCT clause returns only unique rows, removing duplicates from the result set.

It can be applied to one or more columns; in that case, only unique combinations of those column values are kept.

Distinct values in one column

This example returns each city only once from the Customers table. 

SELECT DISTINCT City
FROM Customers;

The number of rows in this result equals the number of different cities where customers live, which is often useful for building filters or dashboards.

Distinct on multiple columns

You can also find unique pairs or combinations, such as country and city together. 

SELECT DISTINCT Country, City
FROM Customers;

Here a row appears only if the (Country, City) pair is unique, which is different from making each column distinct separately.
