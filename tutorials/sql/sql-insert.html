SQL INSERT

The INSERT statement adds new rows to a table by providing values for one or more columns.

You can insert a single row, multiple rows at once, or copy rows from another table using INSERT ... SELECT.

Insert one row

This example inserts a new customer, explicitly listing the target columns in the Customers table. 

INSERT INTO Customers (CustomerID, CustomerName, City, Country)
VALUES (101, 'Alex Patel', 'Mumbai', 'India');

Listing column names makes the statement more robust to future schema changes and avoids mistakes with column order.

Insert multiple rows

Many databases support inserting several rows in a single statement by providing multiple value lists. 

INSERT INTO Customers (CustomerName, City, Country)
VALUES
  ('Maria Gomez', 'Madrid', 'Spain'),
  ('John Smith', 'London', 'UK'),
  ('Sara Khan', 'Delhi', 'India');

Batch inserts reduce round trips to the database and are more efficient than sending separate INSERT statements for each row.
