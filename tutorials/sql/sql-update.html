SQL UPDATE

The UPDATE statement modifies existing rows in a table by changing the values of one or more columns.

A WHERE clause usually restricts which rows are updated; if omitted, every row in the table will be changed.

Update a single row

This example changes the city of one customer identified by its primary key. 

UPDATE Customers
SET City = 'Pune'
WHERE CustomerID = 101;

The condition on CustomerID ensures that only one row is affected; always doubleâ€‘check the WHERE clause before running updates.

Update multiple columns

Several columns can be updated in one statement by separating assignments with commas. 

UPDATE Customers
SET CustomerName = 'Alex P.',
    Country      = 'India'
WHERE CustomerID = 101;

When the condition matches multiple rows, every matching row will be updated, which can be powerful but dangerous if the filter is too broad.
