SQL DELETE

The DELETE statement removes rows from a table based on a filter condition in the WHERE clause.

Without a WHERE clause, all rows in the table are deleted, leaving the structure but no data.

Delete a single row

This example deletes one customer with a specific id. 

DELETE FROM Customers
WHERE CustomerID = 101;

Deleting by primary key is a common pattern because it uniquely identifies the row you want to remove.

Delete multiple rows

The same syntax can remove all rows that match a broader condition, such as all customers from a given country. 

DELETE FROM Customers
WHERE Country = 'Spain';

For bulk deletions, many teams wrap the command in a transaction so they can roll back if the affected row count is higher than expected.
