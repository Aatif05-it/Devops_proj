Java Data Types

Java is statically typed: every variable/parameter must declare type. Two categories: Primitive (8 types, value stored directly) vs Reference (objects like String, arrays).

Primitives faster/smaller memory; reference types have methods.

Primitive Data Types

// Integer types
byte  range = 127;     // -128 to 127 (1 byte)
short age = 25;        // -32,768 to 32,767 (2 bytes)
int   salary = 75000;  // -2^31 to 2^31-1 (4 bytes) DEFAULT
long  population = 8000000000L;  // -2^63 to 2^63-1 (8 bytes)

// Floating point
float price = 19.99f;  // 32-bit precision
double total = 123.456;  // 64-bit precision DEFAULT

// Character
char grade = 'A';      // Unicode 16-bit

// Boolean
boolean isActive = true;  // true/false

Type Conversion

Widening (auto/safe): byte → short → int → long → float → double

int i = 100;
long l = i;  // automatic
double d = l;  // automatic

Narrowing (explicit): double → float → long → int → short → byte

double d = 9.99;
int i = (int) d;  // 9 (truncates)

Reference Types Example

String name = "Java Developer";  // Reference type
int[] numbers = {1, 2, 3};      // Array reference

System.out.println(name.length());  // 15
System.out.println(numbers[0]);     // 1
