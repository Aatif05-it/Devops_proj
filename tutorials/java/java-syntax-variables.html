Variables Complete Guide

Syntax: <type> <name> = <value>; Declare before use, scoped to block {}.

public class Variables {
    public static void main(String[] args) {
        // 1. Declaration & Initialization
        int age = 25;                    // Integer variable
        String name = "John Doe";        // String reference
        double salary = 75000.50;        // Floating point
        final double PI = 3.14159;       // Constant (final)
        
        // 2. Multiple declarations
        int x, y, z;  // Declare only
        x = 10; y = 20; z = x + y;
        
        // 3. Printing variables
        System.out.println("Name: " + name);
        System.out.printf("Age: %d, Salary: %.2f%n", age, salary);
        
        // 4. Scope example
        if(age > 18) {
            String status = "Adult";  // Local to this block
            System.out.println(status);
        }
        // System.out.println(status);  // ERROR: out of scope
    }
}

Naming Conventions

  • camelCase: myVariable, calculateTotal
  • Class names: PascalCase (MyClass)
  • Constants: UPPER_SNAKE_CASE
  • Start with letter/$_ , then letters/digits/$_
  • Avoid reserved keywords: int, class, public, void, etc.

Common Errors

  • variable not initialized → assign value before use
  • incompatible types → match type or cast
  • cannot find symbol → misspelled or wrong scope
