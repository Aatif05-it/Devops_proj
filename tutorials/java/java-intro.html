Hello, World!

public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
        System.out.println("Welcome to Java programming at CodeNexa Academy");
    }
}

Steps: Save as HelloWorld.java, compile javac HelloWorld.java, run java HelloWorld.

Java Versions Timeline

  • Java 1.0 (1996) - Initial release
  • Java 8 (2014) - Lambdas, Streams API
  • Java 11 (2018) - LTS, var keyword, HTTP Client
  • Java 17 (2021) - Current LTS, sealed classes
  • Java 21 (2023) - Latest LTS, virtual threads, pattern matching
  • Java 23 (2024) - Preview features maturing

Download latest LTS from Eclipse Adoptium or Oracle.

Setup Checklist

  • Install JDK (not just JRE)
  • Set JAVA_HOME environment variable
  • Add bin to PATH
  • Verify: java --version & javac --version
  • IDE: IntelliJ IDEA Community (free), VS Code + Extension Pack
